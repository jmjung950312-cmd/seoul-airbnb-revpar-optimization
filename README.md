## 프로젝트 개요

서울 25개 자치구, 32,061개 에어비앤비 리스팅 데이터(2024.10~2025.09)를 분석하여, 호스트가 통제 가능한 변수를 기반으로 RevPAR(Revenue Per Available Room) 최적화 포인트를 탐색한 프로젝트입니다. LightGBM 2-Stage 파이프라인으로 ADR과 예약률을 분리 예측하고, K-Means 클러스터링으로 4개 시장 유형별 차별화 전략을 도출했습니다.

## 분석 파이프라인

```
EDA → 피처 엔지니어링 → 2-Stage ML (ADR + Occupancy) → 클러스터링 → 인사이트 → 대시보드
```

### 2-Stage ML Pipeline

1. **Model A (ADR 예측)**: 자치구 클러스터, POI 거리, 숙소 구성 등 13개 피처 → 시장 적정 1박 요금 예측
2. **Model B (Occupancy 예측)**: 즉시예약, 슈퍼호스트, 리뷰, 사진 + price_gap_oof(OOF 누수 방지) → 예약률 예측
3. **Isotonic Regression**: RevPAR(ADR x 예약률) 단조 증가 보정

### KPI 정의

```python
RevPAR = ADR x Occupancy Rate
revpar_trend = (l90d_revpar - ttm_revpar/4) / (ttm_revpar/4 + 1e-6)

```
